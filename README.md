# 📖 About Repository

**SelfGPT** is a modern AI-powered chatbot and personal assistant web application built using React.js and Vite.

The platform provides an interactive conversational interface where users can communicate with an AI assistant for queries, productivity, learning, and general conversations through a clean and responsive UI.

This project demonstrates frontend AI chatbot integration using modern React architecture and fast development workflows.

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/ERAnkit03/SelfGPT.git
```

---

## 2️⃣ Navigate Into Project Folder

```bash
cd SelfGPT
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Start Development Server

```bash
npm run dev
```
